# 🎨 PixelShare | Muro Colaborativo

Pizarra colectiva en tiempo real donde cualquiera puede dejar su huella. Este proyecto es una demostración técnica de integración **Full-Stack** (Angular + Spring Boot) con persistencia en **MySQL**.

---

### 🚀 Tech Stack

*   **Frontend**: Angular 18 (Standalone), Bootstrap 5, Canvas API.
*   **Backend**: Java 21, Spring Boot 3.4, JPA / Hibernate.
*   **Base de datos**: MySQL 8 (Persistencia total).
*   **Infraestructura**: Docker & Docker Compose (Listo para Portainer).

---

### ✨ Características principales

*   **Lienzo Persistente**: Cada trazo se guarda en el servidor y es visible para todos los usuarios.
*   **Herramientas dinámicas**: Selector de color y ajuste de grosor de pincel en tiempo real.
*   **Diseño Moderno**: Interfaz limpia y responsiva inspirada en herramientas de diseño profesionales.
*   **Docker-Ready**: Configuración completa para despliegue rápido en cualquier VPS.

---

### 🛠️ Instalación rápida

#### Requisitos
*   Java 21+ & Node.js 20+
*   MySQL 8

#### Pasos
1. **Clonar el repo**: `git clone https://github.com/tu-usuario/pixelshare.git`
2. **Backend**: Configura tu usuario/pass de MySQL en `src/main/resources/application.properties` y lanza `DemoApplication.java`.
3. **Frontend**: `cd frontend-demo`, `npm install` y `npm start`.

*(También puedes usar el **`docker-compose.yml`** incluido para desplegar todo en segundos).*
